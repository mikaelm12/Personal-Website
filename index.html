Help! : The Smart Task Manager

Help is a smart task manager thats aides people in managing their time by suggesting when to do what. Users input their tasks and provide information about these tasks such as the date by when it should be completed, the importance of the task and the tasks perceived difficulty of the task. Help then calculates an urgency score for each task and suggests and order in which the users can complete the tastks. The user can also opt to order the tasks by any of the input criteria. Help is available for download on the Google Play Store
